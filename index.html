Software Developement Projects:

Project: Robots vs Dinosaurs

Language: Python

Purpose: Build a program in Python using Visual Studio Code that could have a Herd of three Dinosaurs with Attack Power attributes and a Fleet of three Robots with an importable weapon. Health of individual units was tracked during each turn, and players unit eliminated when health reached zero. Team wins when other team runs out of units. Game will properly display winner and end game.

Project: Random Trip Generator

Language: Python

Purpose: Build a program in Python using Visual Studio Code that allowed a user to randomly generate a trip from user input to different locations. Each selection would then loop into a set of four selections per set. Each set had five sub selections. You could reselect at any point and start over if desired. Program would display completed trip and allow user input to print trip or start over.

Project: Rack, Paper, Scissors, Lizard, Spock

Language: Python

Purpose: Build a program in Python using Visual Studio Code that allowed for Human vs Human or Human vs Ai. Rules were first player to two wins. You had a total of five selections and each selection would beat two of the five selections only. Game tracked wins and would auto select if Ai was opponent. Game would end battle loop and display winner once the win parameters were met.
